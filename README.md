# voxto
turn voice messages into text in seconds.
la app serve a trasformare i messaggi vocali di Whatsapp o altre app in testo e inoltre dà la possibilità di condividere il messaggio ottenuto facilmente. Inoltre ci sarà la funzione di riassumere i messaggi vocali lunghi in un testo breve per poter capire il concetto senza leggere tutto il testo del messaggio.
